# Contributing to LGPD Sentinel AI

Thank you for your interest in contributing to **LGPD Sentinel AI**! This project is 100% open source under Apache 2.0, aimed at automating LGPD compliance audits with AI. We welcome contributions from the community to make privacy tools accessible to everyone.

## How to Contribute

1. **Fork the Repository**: Click the "Fork" button on GitHub and clone your fork locally.

2. **Set Up Environment** (Free Tools Only):
   - Python 3.x (free)
   - Install dependencies: `pip install -r requirements.txt` (all free OSS libs: FastAPI, LangChain, etc.)
   - Use [Hugging Face](https://huggingface.co/) for free open models (e.g., Mistral-7B)
   - For local testing: [Docker](https://www.docker.com/) (free) for self-hosted setup

3. **Create a Branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**: Follow PEP8 code style. Add tests when possible.

5. **Commit and Push**: Use clear commit messages following [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat: Add DPIA audit feature`
   - `fix: Correct data mapping classification`
   - `docs: Update installation guide`

6. **Open a Pull Request**: From your fork to the `main` branch. Describe your changes and reference related issues.

7. **Code of Conduct**: Be respectful and inclusive. Report issues via [GitHub Issues](https://github.com/ldsjunior-ui/lgpd-sentinel-ai/issues).

## Good First Issues

Look for issues labeled [`good first issue`](https://github.com/ldsjunior-ui/lgpd-sentinel-ai/labels/good%20first%20issue) to get started.

## Development Stack (All Free)

- **Backend**: Python + FastAPI
- **AI Models**: LangChain + Ollama + Mistral (Hugging Face, free)
- **Database**: Supabase free tier (PostgreSQL)
- **Frontend**: React OSS
- **Infrastructure**: Docker (self-hosted) + Vercel/Railway (free tiers)
